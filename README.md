# Sentiment Analysis from Scratch (TF-IDF + Naive Bayes)

## Overview
This project implements an end-to-end sentiment analysis system from scratch using core Python.
No NLP or ML libraries such as sklearn, nltk, or spacy were used.

## Features
- Custom tokenizer and stopword removal
- Vocabulary building from training data
- TF-IDF vectorization implemented manually
- Naive Bayes classifier built from scratch
- CSV-based training pipeline
- Flask web application for real-time prediction

## Tech Stack
- Python
- Flask
- Math, CSV, Pickle (standard libraries)

## How to Run
```bash
pip install -r requirements.txt
python train.py
python app.py
